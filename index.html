Replica Example Site
I recreated the samsung website as an example. It features a dynamic navigation bar, media slide section, search interface, and cart interface. Website is also responsive and works on mobile and tablets with an optimized layout.
Matching Card Game
A modern take on the classing matching card game. To win simply match all cards on the board, then a pop up box will appear congratulating the player and showing them their stats from the game and giving them an option to restart.
Frogger Arcade Game
A classic recreation of the popular arcade game frogger. Features enemy characters that move dynamically through the map that the player must avoid. To win, the player must make it across the road to the water using the arrow keys without running into an enemy character.
